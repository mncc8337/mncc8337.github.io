# New game project
15 Dec 2024, 00:30

## What
[We](https://github.com/0x5e8) decided to make a game about growing mold, called Mold Control (what a name).

## The game
The goal of the game would be becoming the mother of mold: collecting all species of mold and all infomation about them. It may sounds bizzare but i think it may be interesting. We got this idea because we have been making cheese recently, and one of the important processes of it is getting good mold to grow on the cheese to develop new flavours. We extended this idea to growing all type of mold to get money to grow our own mold company!
This game will feature a grid placing system. All the building need to be placed manually and must be linked by some ways to get them to work. There would be a lab to research, a farm to grow mold, power generator, and so on. These building can be upgraded from tier 1 to 5 to improve their productivity. There are also some decorative like trees, poles, pavements to decorate the farm
It would be pretty boring if there are no special events, so i think it need some kind of natural disaster like mold spread, breeze or flaming weather, etc. Or maybe some kind of special mold that only appear in a specific amount of time like luminescense mold? If i want to implement all of these i would probably need a datetime system and a day-night cycle.
I hope this project will be complete (and not being dumped into dumpster like my other projects lol).
